Chief Data Architect at Ontotext (Q7095072): software company.

  • Wikidata: Vladimir Alexiev (Q48975668).
  • Publications: https://rawgit2.com/VladimirAlexiev/my/master/index.html

Working on LOD and semantic modeling:

  • cultural heritage, eg Nomenclature for Museum Cataloging (Q64351785): structured vocabulary for object naming and classification, maintained by Canadian Heritage Information Network (CHIN) and American Association for State and Local History (AASLH)
  • authority control, eg P7859 (P7859) and VIAF
  • science, eg Tracking of Research Results (Q56259739): EC DG RTD project to track the impact of all FP7-Collaboration projects
  • company data, eg GLEIF registration authority code (P9487), OpenCorporates register ID (P9532), OpenCorporates register jurisdiction (P9630), MIC market code (P7534) (see Wikidata_talk:WikiProject_Companies#More_Registers
  • geographic data, eg BG EKATTE place ID (P3990)
  • Contributing not just prop definitions but large amounts of data: I've added data to 407 different properties?! See the rightmost Userbox "Contributions across wikis"
