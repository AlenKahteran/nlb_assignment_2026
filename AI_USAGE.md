# AI Usage

AI coding assistance was used to draft and implement the project scaffold, CLI structure, SQLite schema, extraction flow, tests, and documentation. Review of the result was done at every step, especially for benchmark outputs and source-quality assessment.

Harness used was `Codex` from OpenAI. 
The LLM model used with the harness was `GPT-5.5`. 
Model used for anwsering the questions was `gpt-5.4-mini` for latency and cost reasons.
